Resource Records for DNSSEC

The following resource records are implemented for DNSSEC

  • DNSKEY Public Key
  • DS Delegation Signer
  • NSEC Next Secure
  • NSEC3 Authenticated Next Secure
  • NSEC3PARAM Authenticated Next Secure Parameters
  • OPT Extension mechanism (EDNS)
  • RRSIG Signature for a set of resource records

Message Headers Flags

The following message header flags are defined

  • AD Authenticated Data
  • CD Checking Disabled
  • DO DNSSEC OK EDNS header bit
