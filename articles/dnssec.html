DNSSEC

DNSSEC RFC4035 provides data integrity and authentication to security aware resolvers and applications through the use of cryptographic digital signatures.

DNSSEC – What Is It and Why Is It Important?

Requesting DNSSEC

The client sends a DNS query with the DO bit set. The server will then responsd with the answers and also the RRSIGs that are used to validate the answers.

// Request the IPv6 addresses for "example.com"
// RD is "recursion desired"
// UseDnsSecurity() enables signed answers.
var query = new Message()
{
    RD = true,
    Questions =
    {
        new Question { Name = "example.com", Type = DnsType.AAAA }
    }
}
    .UseDnsSecurity();

// Get an answer.
var response = await resolver.ResolveAsync(query);
