TTL

Specifies the time interval that the resource record may be cached before the source of the information should again be consulted.

Declaration
public TimeSpan TTL { get; set; }
Property Value
Type Description
TimeSpan

Defaults to 1 day.
Remarks

Zero values are interpreted to mean that the RR can only be used for the transaction in progress, and should not be cached.

For example, SOA records are always distributed with a zero TTL to prohibit caching. Zero values can also be used for extremely volatile data.
